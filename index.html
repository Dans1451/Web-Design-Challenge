A Comparison of Weather to a City's Location

Summary: Latitude vs. X
Visualizations

This is a project comparing a city's latitude with the max measured temperature, humidity, cloudiness and windspeed for the day of 1/05/2017 This was accomplished by pulling data form OpenWeatherMap API and creating a dataset of over 500 cities. Matplotliob was then used to create visualizations of the these relationships. This site shows those relationships.
